# Portfólio - Henrique Palermo

Este repositório contém o código do meu portfólio pessoal, desenvolvido para apresentar meus projetos, habilidades e experiência como desenvolvedor.

O objetivo do site é centralizar meus principais trabalhos e tecnologias utilizadas no desenvolvimento de aplicações web modernas.

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- React
- JavaScript
- Bootstrap
- styled-components
- Vite

## Funcionalidades

- Apresentação pessoal
- Listagem de habilidades
- Exibição de projetos com:
  - link para GitHub
  - link para demo
- Layout responsivo
- Componentização com React
